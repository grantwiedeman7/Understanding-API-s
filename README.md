**Author:** Grant Wiedeman

## Tutorial Summary

This tutorial introduces the concpet of Application Programming Interfaces (APIs) in a simple and practical way. You will learn the definiton and basics of what an API is along with some example and a walkthrough of creating your first API! By the end, you will understand how modern apps communicate with each other, how data travels across the web, and how APIs power everything from weather apps to online shopping.

---

## Target Audience

This tutorial is desinged for:

-Students in introductory IT or computing courses
-Anyome new to programming, data, or web technologies
-Peopple with little to no prior knowledge of APIs
-People understand basic web concpets and file usage but not the backgorund works of APi requests

*Technical experience helps but is not required to understand and work in this tutorial
